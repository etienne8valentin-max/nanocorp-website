## Notes de production

### Conseils généraux TikTok
- **Hook < 3 secondes** : texte à l'écran DÈS la première frame, pas d'intro logo
- **Rythme** : cut toutes les 3–5 secondes max sur la partie corps
- **Sous-titres** : toujours (60% des TikTok regardés sans son)
- **Format** : 9:16 vertical, 1080×1920 px
- **Durée optimale** : 40–50 secondes (meilleur ratio rétention/complétion)

### Calls-to-action
- Lien bio → https://spiregg.nanocorp.app
- Code promo à tester : `LAUNCH30` (-30% affiché sur /cart)

### Ordre de tournage recommandé
1. Script 7 (EN — max de portée internationale)
2. Script 5 (valeur — meilleur potentiel viral FR)
3. Script 1 (démo — core content)
4. Scripts 2, 3, 6 (urgence, before/after, frustration — high relatability)
5. Scripts 8, 9, 10 (niches — audience ciblée)
6. Script 4 (comparatif — requiert un peu plus de montage)
